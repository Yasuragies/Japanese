Pointing + Topic particle

Required vocabulary:
これ : this
それ : that
あれ : that (over there)
かさ : umbrella

これ・それ・あれ can be combined with Topic particle (は) which result in:
これはB (これ is used when the object is close to the speaker)
それはB (それ is used when the object is close to the listeners)
あれはB (あれ is used when the object is far from both speaker and listeners)

Example:
これ は ペン
this is pen

それ は かばん
that is bag

あれ は かさ
that over there is an umbrella
