Pointing ＋ の
Required vocabulary
この = this
その ＝ that
あの = that (over there)
車 = くるま = Chinese
さいふ = ひと = person
くつ ＝ にほんじん = Japanese (person)
学校 = ちゅうごくじん = Chinese (person)
人 = ひと = person
時計 = とけい = watch
パソコン = computer
かのじょ = girlfriend (as noun) / she/her (as pronoun)
かれし = boyfriend (noun)
かれ = he/him (pronoun

Grammar
 この B
This B

Example

こ の ぺん
This pen

そ の 車
that car.

この 人 は たけし です
This person is Takeshi

あの 人 は たけし の かのじょ です
That person is Takeshi’s girlfriend

この かばん は たけし の かばん です
That bag is Takeshi’s bag

あの時計はたけしのかのじょの時計です。
That watch is Takehi’s girlfriend watch
