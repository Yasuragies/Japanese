になる・to become
        になる・to become

        Grammar
        になる means to become, it indicates changes
        it can be used on 
        noun + になる。
        な-adj + になる。

        医者になる
        to become a doctor

        好きになる to become like (to grow fond of..)

        to use なる form in い-adjective
        replace い with くなる

        かわいい -> かわいくなる
        cute -> to become cute.

        さびしい -> さびしくなる
        lonely -> to become lonely.

        かなしくなる = to become sad
        うれしくなる = to become happy
        うるさくなる = to become noisy
        やさしくなる = to become kind


        なる itself is a godan verb. 
        
        ならない = not to become
        なりたい = want to become.
        なりたくない = don't want to become.
        になった = became.

        弁護士にならい
        not becoming a lawyer.

        かわいくなりたい
        (I) want to become right.

        さびしくなりたくない
        (i) don't want to become lonely.

        好きになった
        became like/fond.

        every conjugation that ended with い can be replaced with かった in order to turn it into past adjective

        ならなかった = didn't become
        なりたかった = wanted to become
        なりたくなかった = didn't want to become.
