に particle
Required vocabulary
 行く = いく
寝る = ねる = to sleep
帰る = かえる = to return home
駅 = えき = station
学校 = school
図書館 = library
きっさてん = café
郵便局 = post office
レストラン = restaurant
~時 = じ = O’clock (suffix) ex １時
午前 = p.m
午後 = a.m
昨日 = きのう = yesterday
今日 = きょう = today
明日 = あした = tomorrow
金曜日 = きんようび = Friday.
来年 = らいねん = next year.
来週 = らいしゅう = next week.
ある　= to exist (object)
いる = to exist (living being)
ここ = here
そこ = there
あそこ = over there (far)

に as destination particle.
A に B
Noun に verb

in this case に mark A (noun) as the destination of our action (B)

駅に行く
(I) (will) go to the station

きっさてん に　行く
(I) (will) go to the café


に to mark location
駅 は ここ に ある
the station is here.

たけし先生 は 学校 に いる
Takeshi sensei is in school.

かのじょ は あそこ に いる
she is over there.

に to mark time
Example

九時(くじ) に 寝る
（I）(will) sleep at 9'clock

金曜日 に 図書館 に 行く
(I will) go to library at friday

Particle に cannot be used on relative time particle such as

今日 = きょう
来週 = らいしゅう
来年　= らいねん

thus it is followed by topic particle は

今日はきっさてんに行く
today (I) will go to the cafe.

when we want to set the topic, time can follows after the topic

ふみこ は 明日、がっこう に　行く
Fumiko　(will) go to school tommorow

けん は 来週、映画 を 見る
Ken (will) watch a movie next week
