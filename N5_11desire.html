Desire conjugation

Required vocabulary
~ichidan~
 食べる = たべる = to eat
見る = みる = to see/watch
寝る = ねる = to sleep
起きる = おきる = to wake up

~ichidan~
 会う = あう = to meet
待つ = まつ = to wait
帰る = かえる = to return home
読む = よむ = to read
遊ぶ = あそぶ = to play
死ぬ = しぬ = to die
聞く = きく = to listen
行く = いく = to go

irregular verbs
 する = to do
やる = to do

~noun~
Grammar
Desire form is used if you want to express what you want to and what you don't want to do

Ichidan verb Desire form conjugation.
to turn ichidan verb into Desire form, replace the る in the end of the verb with たい

食べる -> 食べたい
eat -> want to eat

見る ->　見たい
see -> want to see

寝る -> 寝たい
sleep -> want to sleep

起きる -> 起きたい
wake up -> want to wake up

Godan verb Desire form conjugation
conjugate う・つ・る　to Desire form.
 to conjugate this category into Desire form, replace the　う・つ・る part。
う -> いたい
つ -> ちたい
る -> りた

会う -> 会いたい
meet -> want to meet

待つ -> 待ちたい
wait -> want to wait

帰る -> 帰りたい
return home -> want to return home

conjugate む・ぶ・ぬ　to　to Desire form.
 to conjugate this category into Desire form, replace the　む・ぶ・ぬ part。
む -> みたい
ぶ -> びたい
ぬ -> にたい

読む -> 読みたい
read -> want to read

遊ぶ -> 遊びたい
play -> want to play

死ぬ -> 死にたい
die -> want to die

conjugate く　into　to Desire form.
 to conjugate く to Desire form, replace く with きたい

聞く　-> 聞きたい
hear -> want to hear

行く　-> 行きたい
go -> want to go

conjugate す　into Desire form.
 to conjugate す to Desire form, replace す with したい

話す　-> 話したい
speak -> want to speak

Irregular verb Desire form conjugation
 する -> したい
やる -> やりたい

Example:

料理したい
want to cook.

掃除したい
want to clean.

勉強したい
want to study.

turning Desire form into negative Desire. After we conjugate a verb to Desire form replace the い with くない. this rule works for Ichidan, Godan, and Irregular.

Ichidan
食べたい -> 食べたくない
見たい -> 見たくない
寝たい -> 寝たくない
起きたい -> 起きたくない

Godan

会たい -> 会いたくない
want to meet -> don't want to meet

待ちたい -> 待ちたくない
want to wait -> don't want to wait

帰りたい -> 帰りたくない
want to return home -> don't want to return home

読みたい -> 読みたくない
want to read -> don't want to read

遊びたい -> 遊びたくない
want to play -> don't want to play

死にたい -> 死にたくない
want to die -> don't want to die

聞きたい -> 聞きたくない
want to hear -> don't want to hear

行きたい -> 行きたくない
want to go -> don't want to go

話したい -> 話したくない
want to speak -> don't want to speak

Irregular

したい -> したくない
want to do -> don't want to do

やりたい -> やりたくない
want to do -> don't want to do
