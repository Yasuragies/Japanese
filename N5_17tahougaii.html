        ほうがいい・Planning to do
        
        Grammar
            ほうがいい means "its better if you~"
            this gammar is used to give advice to someone
            
            Verb た form + ほうがいい

            食たべたほうがいい
            It's better if you eat

            薬くすりを飲のんだことほうがいい
            It's better if you drink medicine

            医者いしゃへ行いったほうがいい
            It's better if you go to a doctor

            this grammar can also be used for for negative (it's better if don't..)

            ふみこ の 料理りょうり を 食たべないほうがいい
            It's better if you don't eat fumiko's cookings

            この酒さけを飲のんだことほうがいい
            It's better if you drink this sake.

            あの人いしゃと話はなさないほうがいい
            It's better if you don't talk to that person.
