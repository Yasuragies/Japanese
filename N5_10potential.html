Grammar
Potential form is used if you want to express "I can/can't" as in
"I can't eat that"

Ichidan verb potential form conjugation.
to turn ichidan verb into potential form, replace the る in the end of the verb with られる

食べる -> 食べられる
eat -> can eat

見る ->　見られる
see -> can see

寝る -> 寝られる
sleep -> can sleep

起きる -> 起きられる
wake up -> can wake up

Godan verb potential form conjugation
conjugate う・つ・る　to potential form.
 to conjugate this category into potential form, replace the　う・つ・る part。
う -> える
つ -> てる
る -> れる

会う -> 会える
meet -> can meet

待つ -> 待てる
wait -> can wait

帰る -> 帰れる
return home -> can return home

conjugate む・ぶ・ぬ　to　to potential form.
 to conjugate this category into potential form, replace the　む・ぶ・ぬ part。
む -> める
ぶ -> べる
ぬ -> ねる

読む -> 読める
read -> can read

遊ぶ -> 遊べる
play -> can play

死ぬ -> 死ねる
die -> can die

conjugate く　into　to potential form.
 to conjugate く to potential form, replace く with ける

聞く　-> 聞ける
hear -> can hear

行く　-> 行ける
go -> can go

conjugate す　into potential form.
 to conjugate す to potential form, replace す with せる

話す　-> 話せる
speak -> can speak

Irregular verb potential form conjugation
 する -> できる
やる -> やれる

Example:

料理できる
can cook.

掃除できる
can clean.

勉強できる
can study.

turning potential form into negative potential. After we conjugate a verb to potential form replace the る with ない. this rule works for Ichidan, Godan, and Irregular.

Ichidan
食べられる -> 食べられない
見る -> 見られない
寝る -> 寝られない
起きる -> 起きられない

Godan

会える -> 会えない
can meet -> can't meet

待てる -> 待てない
can wait -> can't wait

帰れる -> 帰れない
can return home -> can't return home

読める -> 読めない
can read -> can't read

遊べる -> 遊べない
can play -> can't play

死ねる -> 死ねない
can die -> can't die

聞ける -> 聞けない
can hear -> can't hear

行ける -> 行けない
can go -> can't go

話せる -> 話せない
can speak -> can't speak

Irregular

できる -> できない
can do -> can't do

やれる -> やれない
can do -> can't do
