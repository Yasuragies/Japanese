て form conjugation

て Form
Grammar
 There are many usage of て form, but in this page we're going to learn how to use て form as a way to request someone to do something for you.
but before we start, let us learn how to conjugate normal verb into て form.

Ichidan
 To turn ichidan verb into て form, replace the る with て 食たべる -> 食たべて
eat -> please eat

見みる ->　見みて
see -> please see

寝ねる -> 寝ねて
sleep -> please sleep

起おきる -> 起おきて
wake up -> please wake up

Godan verb て form form conjugation
conjugate う・つ・る　to て form form.
 replace the　う・つ・る part with って。
う -> って
つ -> って
る -> って

会う -> 会って
meet -> please meet

待つ -> 待って
wait -> please wait

帰る -> 帰って
return home -> please return home

conjugate む・ぶ・ぬ　to　to て form form.
 replace the　む・ぶ・ぬ part with んで。
む -> んだ
ぶ -> んだ
ぬ -> んだ

読む -> 読んで
read -> please read

遊ぶ -> 遊んで
play -> please play

死ぬ -> 死んで
die -> please die

conjugate く　into　to て form form.
 replace く with きたい

聞く　-> 聞いて
listen -> please listen

Exception just for 行く
 Replace く with って

行く - 行って
go -> please go

conjugate す　into て form form.
 replace す with して

話す　-> 話して
speak -> please speak

Irregular verb て form form conjugation
 する -> して
やる -> やって

Example:

料理して
please cook.

掃除して
please clean.

勉強して
please study.
