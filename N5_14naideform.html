ないで form conjugation
        ないで Form
        
        Grammar
            ないで form is used to request someone not to do someth.
            
            Ichidan
            To turn ichidan verb into  form, replace the る with ないで
            食たべる -> 食たべないで
            eat -> (please) don't eat
            
            見みる ->　見みないで
            see/watch -> (please) don't see/watch
            
            寝ねる -> 寝ねないで
            sleep -> (please) don't sleep
            
            起おきる -> 起おきないで
            wake up -> (please) don't wake up
            
            Godan verb ないで form form conjugation
            
            conjugate う・つ・る　to ないで form form.
            replace the　う・つ・る part
            う -> わないで
            つ -> たないで
            る -> らないで
            
            会あう -> 会あわないで
            meet -> (please) don't meet
            
            待まつ -> 待またないで
            wait -> (please) don't wait
            
            帰かえる -> 帰かえらないで
            return home -> (please) don't return home
            
            conjugate む・ぶ・ぬ　to　to ないで form form.
            replace the　む・ぶ・ぬ part。
            む -> まないで
            ぶ -> ばないで
            ぬ -> なないで
            
            読よむ -> 読よまないで
            read -> (please) don't read
            
            遊あそぶ -> 遊あそばないで
            play -> (please) don't play
            
            死しぬ -> 死しなないで
            die -> (please) don't die
            
            conjugate く　into　to ないで form.
            replace く with かないで
            
            聞きく　-> 聞きかないで
            listen -> (please) don't listen
            
            行いく - 行いかないで
            go -> (please) don't go
            
            conjugate す　into ないで form form.
            replace す with しないで
            
            話はなす　-> 話はなしないで
            speak -> (please) don't speak
            
            Irregular verb ないで form form conjugation
            する -> しないで
            やる -> やらないで
            
            Example:
            
            料理りょうりしないで
            (please) don't cook.
            
            掃除そうじしないで
            (please) don't clean.
            
            勉強べんきょうしないで
            (please) don't study.
